🧠 Engenharia de Prompts e Cicatrizes de Aprendizado
Nesta etapa, o foco foi aprender a perguntar melhor para a IA, testando variações de prompts e analisando a qualidade das respostas.
🔎 Prompt 1 – Conceito Básico
Explique o que faz um Analista de Dados Júnior e quais são suas principais responsabilidades.

✅ Resultado:
Resposta clara, porém genérica.
🛠️ Ajuste realizado:
Solicitei exemplos práticos e ferramentas.

🔎 Prompt 2 – Versão Refinada
Explique o papel de um Analista de Dados Júnior, incluindo exemplos do dia a dia e as principais ferramentas utilizadas.

✅ Resultado:
Resposta mais completa, com menção a Excel, SQL, Python e dashboards.
📌 Referência cruzada:
IBM + Microsoft Learn

🔎 Prompt 3 – Análise Exploratória
Explique o que é Análise Exploratória de Dados (EDA) e por que ela é importante para iniciantes.

⚠️ Dificuldade encontrada:
Resposta muito técnica.
🛠️ Solução (Prompt melhorado):
Explique Análise Exploratória de Dados como se fosse para alguém iniciando na área de dados, com exemplos simples.

✅ Aprendizado:
A IA responde melhor quando o nível de conhecimento esperado é explicitado.

🧩 Principais Cicatrizes (Troubleshooting)

Prompts genéricos → respostas superficiais
Falta de contexto → respostas confusas
Melhor desempenho quando:

Defini o público-alvo
Pedi exemplos práticos
Solicitei comparações ou listas

✅ Resumo Estruturado
📊 O que é Análise de Dados?
Processo de coletar, limpar, analisar e comunicar dados para apoiar decisões.
👩‍💻 Analista de Dados Júnior
Profissional em início de carreira que:

Analisa dados estruturados
Cria relatórios e dashboards
Apoia áreas de negócio com insights

🧰 Ferramentas Mais Comuns

Excel / Google Sheets
SQL
Python (Pandas, NumPy)
Power BI / Tableau


📖 Glossário Essencial

| Termo | Definição |
|------|----------|
| Dados Estruturados | Dados organizados em linhas e colunas, como planilhas e tabelas de banco de dados |
| EDA | Análise Exploratória de Dados, usada para entender padrões e identificar problemas nos dados |
| KPI | Indicador-chave de desempenho usado para medir resultados |
| Dataset | Conjunto de dados utilizados para análise |
| Dashboard | Painel visual que apresenta indicadores e métricas |
